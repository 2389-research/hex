# Hex Help

## Available Slash Commands

Type `/` followed by a command name to use these workflows:

| Command | Description |
|---------|-------------|
| `/help` | Show this help information |
| `/plan` | Create a detailed implementation plan with bite-sized tasks |
| `/brainstorm` | Explore ideas and design through collaborative dialogue |
| `/review` | Review code changes with systematic analysis |
| `/debug` | Systematic debugging workflow for investigating issues |
| `/test` | Test-Driven Development workflow |
| `/commit` | Generate and review commit messages |
| `/refactor` | Code refactoring workflow with safety checks |
| `/document` | Generate documentation for code |

## Keyboard Shortcuts

### Navigation
| Key | Action |
|-----|--------|
| `↑/↓` | Scroll viewport / Navigate autocomplete |
| `Tab` | Show autocomplete / Accept suggestion |
| `Esc` | Close autocomplete / Cancel action |
| `?` | Toggle help panel |

### Actions
| Key | Action |
|-----|--------|
| `Enter` | Send message |
| `Alt+Enter` | New line in input |
| `Ctrl+C` | Cancel stream / Quit (press twice) |
| `Ctrl+K` | Clear conversation |
| `Ctrl+L` | Clear screen |
| `Ctrl+O` | Toggle tool output log |
| `Ctrl+S` | Save conversation |
| `Ctrl+E` | Export conversation |

### Tool Approval
| Key | Action |
|-----|--------|
| `↑/↓` | Select option |
| `Enter` | Confirm selection |
| `y` | Approve (quick) |
| `n` | Deny (quick) |

## Tips

- Type `/` to see all available commands in autocomplete
- Commands can have arguments: `/plan authentication system`
- Use `Ctrl+O` to see full tool output when collapsed
- History navigation: `↑/↓` in empty input field

For more information, visit: https://github.com/2389-research/hex
